# TapClick (自动跳过广告)

#### 基于Android无障碍服务开发的应用，可以实现但不局限于以下的功能，更多的功能可通过创建规则实现:

① 自动跳过开屏广告；

② 自动关闭弹窗；

③ 自动关闭广告；

④ ......

本软件需要长驻后台，但所占内存和耗电量极少，一般不会影响体验；
